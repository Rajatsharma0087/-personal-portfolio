# 💼 Personal Portfolio - Rajat Sharma

A modern, responsive portfolio website showcasing my web development skills and projects.

## 🌐 Live Demo
👉 **[View Portfolio](https://rajatsharma0087.github.io/personal-portfolio/)**

## ✨ Features

- **Responsive Design** - Works on all devices (mobile, tablet, desktop)
- **Modern UI** - Clean, professional dark theme
- **Projects Showcase** - Display of my work with live links
- **Skills Section** - Technologies and tools I use
- **Contact Form** - Easy way to get in touch
- **Smooth Scrolling** - Enhanced user experience

## 🛠️ Built With

- **HTML5** - Semantic markup
- **CSS3** - Modern styling
  - Flexbox
  - CSS Grid
  - Media Queries
  - Animations
- **Responsive Design** - Mobile-first approach

## 📋 Sections

1. **Hero/Header** - Introduction with call-to-action
2. **About** - My story and background
3. **Skills** - Technologies I work with
4. **Projects** - Portfolio of my work
5. **Contact** - Get in touch form and links
